alias:: C++/语句

- ^^C++/语句^^是[[C++ 程序]]的组成部分，按顺序执行。
	- 如任何 函数体 都是语句的序列。
	  ``` cpp
	  int main()
	  {
	      int n = 1;                        // 声明语句
	      n = n + 1;                        // 表达式语句
	      std::cout << "n = " << n << '\n'; // 表达式语句
	      return 0;                         // 返回语句
	  }
	  ```
